Query parameter parser and dumper

Send query parameters and see how they parse. This tool was built to help build confidence in a job application puzzle I built a number of years ago. The puzzle judges you harshly for even a single failure, so be sure of the information you're sending, and only send it once.

This tool runs entirely within your browser, but your query may appear in server logs.

Special parameters

Use these special keys to trigger parsing with a certain type

  • json: Parse as JSON
  • int, number: Parse as an integer
  • float: Parse as a floating point number
  • bool: Parse as boolean
